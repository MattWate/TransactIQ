# TransactIQ Pexels Image Mapping

Copy these files into:

`transactiq-site/assets/images/`

## Recommended usage

- `home-hero-advisory-meeting.jpg` — Homepage hero. Strongest replacement for the current AI-style glowing handshake.
- `data-portfolio-dashboard.jpg` — Supporting data-led insight / analytics background. Use sparingly because it can feel trading-specific.
- `market-performance-chart.jpg` — Supporting performance/risk/portfolio background. Best for a dark overlay section, not a main hero.
- `about-professional-handshake.jpg` — About page hero or final CTA. Clean, human and professional.
- `consumer-payment-card.jpg` — Consumer Consulting hero or consumer payment card section. Simple and payment-specific.
- `consumer-card-transaction.jpg` — Consumer Consulting or payment transaction section. Very relevant, more operational/real-world.
- `careers-corporate-architecture.jpg` — Careers or About hero. Abstract, corporate and premium with a navy overlay.

## Suggested CSS swaps

```css
/* Homepage hero */
url("../images/home-hero-advisory-meeting.jpg")
```
